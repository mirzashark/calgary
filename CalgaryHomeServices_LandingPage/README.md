# Calgary Home Services - Landing Page

A highly polished, responsive, and modern landing page for **Calgary Home Services** specializing in Furnace and Air Duct Cleaning.

## Features

- **Responsive Design**: Mobile-first architecture with a custom CSS hamburger drawer menu.
- **Interactive Before/After Slider**: Dynamic visual proof component showcasing duct cleaning results using `clip-path` and keyframe-synchronized animations.
- **Conversion Optimization**: High-converting hero section with a dispatch booking form, trust metrics, custom value cards, and a neighborhood service lookup.
- **Brand Consistency**: Fully styled using the corporate blue palette matching the official CHS logo.

## Assets Included

- `index.html` - Core HTML & Styling logic.

- `Media/After.png` - After state for the interactive slider.

## How to Run Locally

You can open `index.html` directly in any web browser. 

For local development with hot-reloading:

1. Install Node.js if not already installed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm start
   ```
